Tic-Tac-Toe-Protocol

Summary

Tic-Tac-Toe Protocol is a Python project via the console that is started up as a server hosting up to two players (clients). The game is based around using TCP, where it simulates the three-way handshake model. Players would send commands to the server, such as input actions. Then the server would acknowledge the given commands and decide to update the game state under certain conditions. Once the players received the server acknowledgement, the player would finish the process by sending an ackowledge packet back to the server.
